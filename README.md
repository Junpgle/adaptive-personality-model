# 我的相处模型｜自适应行为问卷

这是一个无需服务器即可运行的单页网页。下载后直接双击 `index.html` 即可使用，也可以部署到 GitHub Pages、Vercel 或 Netlify。

## 当前版本

- 100 道互不重复的日常情境题
- 7 个观察维度
- 精简、标准、深入三种自适应模式
- 每个维度独立计算回答数量、一致性、正反向题覆盖和置信度
- 某个维度达到高置信度后立即停止追问
- 低置信度维度会继续追加不同情境题
- 深入模式在必要时最多可使用完整 100 题题库
- 自动生成雷达图、逐维度解释和 Markdown 模型
- 答案仅保存在浏览器本地，不上传服务器

## 题目分布

- 内在驱动：15 题
- 稳定交付：15 题
- 反馈接纳：14 题
- 互惠共情：14 题
- 清晰沟通：14 题
- 公共责任：14 题
- 边界成熟：14 题

合计：100 题。

## 使用边界

本工具用于自我反思，不是经过临床或心理测量学验证的诊断量表。结果应结合真实行为、他人反馈和后续变化持续修正，不建议用于审判、羞辱或给他人贴标签。
